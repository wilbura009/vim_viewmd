# vim_viewmd

A vim plugin for the [viewmd](https://github.com/wilbura009/gtk4_playground/blob/master/proj/viewmd/README.md) application.

## Installation

VimPlug

1. Add to `.vimrc`

    ```txt
    Plug https://github.com/wilbura009/vim_viewmd.git
    ```

2. In vim 

    ```txt
    :PlugInstall
    ```
